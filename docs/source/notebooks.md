# Notebooks

The repository includes Jupyter notebooks under `notebooks/` as portfolio artifacts and hands-on examples.

## Included notebooks

- `01_repo_overview.ipynb`: a guided introduction to the project, architecture, repository layout, and provider strategy
- `02_usage_examples.ipynb`: a practical tutorial covering setup, Streamlit usage, CLI commands, Python API usage, and troubleshooting

These notebooks are meant to read like tutorials, not placeholders.
